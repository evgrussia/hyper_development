# Information Architecture

## Content Inventory

| Тип контента | Источник | Где отображается |
|--------------|----------|-------------------|
| Герой (заголовок, подзаголовок, CTA) | Админка / секции лендинга или конфиг | Лендинг, верх экрана |
| Услуги (название, описание, порядок) | Админка, CRUD «Услуги» | Секция «Услуги» |
| Портфолио (кейсы: название, описание, ссылка, изображение, порядок) | Админка, CRUD «Портфолио» | Секция «Портфолио» |
| О разработчике (текст, факты) | Админка / секции лендинга | Секция «О разработчике» |
| Модули заказа (название, описание, порядок) | Админка, CRUD «Модули заказа» | Форма заказа (чекбоксы/мультиселект) |
| Заказы (контакт, описание, выбранные модули, статус, дата) | Бэкенд, создаются с формы; просмотр в админке | Админка «Заказы»; уведомления в Telegram |
| Секции лендинга (вкл/выкл, порядок) | Админка «Секции лендинга» | Определяет порядок и видимость блоков на лендинге |

## Site Map

### Публичная зона (лендинг)

```
hyper-development.ru/
├── / (одностраничный лендинг)
│   ├── #hero          — Герой
│   ├── #services      — Услуги
│   ├── #portfolio     — Портфолио
│   ├── #about         — О разработчике
│   └── #order         — Форма заказа
└── (внешние ссылки из портфолио/футера — опционально)
```

Одна страница; навигация — якорные ссылки и/или скролл.

### Админка

```
/admin/  (или /manage/, префикс по решению)
├── /login             — Вход
├── /portfolio         — Список кейсов, CRUD
├── /services          — Список услуг, CRUD
├── /sections          — Секции лендинга (порядок, вкл/выкл)
├── /order-modules     — Модули заказа, CRUD
└── /orders            — Список заказов, детали, статусы
```

## Navigation Model

### Лендинг

- **Primary:** якорные ссылки в шапке (Услуги, Портфолио, О разработчике, Заказать) — фиксированная или скрытая шапка при скролле.
- **Secondary:** CTA в герое («Заказать» / «Выбрать модули»); повтор CTA перед/в блоке формы.
- **Utility:** опционально — ссылка на Telegram, соцсети в футере.

### Админка

- **Primary:** боковое меню или горизонтальные табы — Портфолио, Услуги, Секции, Модули заказа, Заказы.
- **Utility:** Выход (логаут).

## Labeling System

| Элемент | Рекомендуемая подпись (RU) |
|--------|----------------------------|
| Герой CTA | «Заказать проект» / «Выбрать модули» |
| Секция услуг | «Услуги» |
| Секция кейсов | «Портфолио» / «Примеры работ» |
| Секция о себе | «О разработчике» |
| Форма заказа | «Оставить заявку» / «Заказать» |
| Поле контакта | «Email» / «Телефон» (или «Контакт для связи») |
| Модули в форме | «Выберите модули» / «Модули проекта» |
| Кнопка отправки | «Отправить заявку» |
| Админка: разделы | «Портфолио», «Услуги», «Секции лендинга», «Модули заказа», «Заказы» |
| Статусы заказа | «Новый», «В работе», «Выполнен», «Отменён» |

## Search Strategy

Поиск на сайте не предусмотрен (одностраничный лендинг + админка с небольшим числом разделов). При росте контента портфолио можно добавить фильтр/поиск в админке в разделе «Заказы».
