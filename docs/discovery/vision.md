# Vision: Сайт агентства Hyper-Development

## Problem Statement

Заказчикам web-приложений (стартапы, бизнес, грантодержатели) сложно быстро оценить компетенции разработчика и оформить заказ в одном месте. Нужна единая точка входа: портфолио с примерами работ и прозрачная форма заказа с выбором модулей.

## Value Proposition

**Для заказчика:** один сайт — портфолио, описание услуг, заказ проекта с выбором модулей и уведомлением в Telegram.  
**Для владельца:** админка для управления портфолио, услугами, секциями лендинга и заказами; все заказы приходят в Telegram webapp.

## Target Audience (Personas)

| Persona | Описание | Цель на сайте |
|--------|----------|----------------|
| **Заказчик MVP** | Основатель/продукт, нужен быстрый MVP | Увидеть кейсы, оформить заказ с выбором модулей |
| **Бизнес-заказчик** | Нужна доработка/рефакторинг или интеграции | Понять услуги, оставить заявку |
| **Грантодержатель** | Нужна документация и реализация под грант | Узнать про 24ч-пакет и грантовое сопровождение |

## Success Metrics (KPIs)

- Конверсия посетитель → отправка заказа (целевая базовая — измерить после запуска).
- Количество заказов в месяц.
- Время до первого ответа на заказ (через Telegram).

## High-Level Solution

1. **Публичный лендинг** (hyper-development.ru): герой, услуги, портфолио, «О разработчике», форма заказа с выбором модулей (модули из админки).
2. **Отправка заказа:** данные уходят в бэкенд, уведомление — в Telegram bot (webapp для владельца).
3. **Админка:** управление портфолио, услугами, секциями лендинга, модулями заказа и заказами. Язык интерфейса — RU.

## Competitive Advantage

- Позиционирование: AI-агентская система, быстрый выход в продакшен, качественная архитектура (DDD, Clean).
- Уникальное предложение: за 24ч — документы + промо + демо; модульный заказ из каталога модулей агентской системы.
- Уведомления только через Telegram — минимум отвлечений, всё в одном мессенджере.

## Constraints (из решений)

- Домен: **hyper-development.ru**, хостинг: **VPS 213.159.67.199**.
- Локаль: **RU**.
- Стек: **Vite + React** (фронт), **Django** (бэкенд).
- Дизайн: современный трендовый **2026**.
- Модули заказа: хранятся и редактируются в **админке**.
- Уведомления о заказах: только **Telegram bot (webapp)**.
